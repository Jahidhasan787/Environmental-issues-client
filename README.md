
Website Name: EcoFixr
Live Link: https://subtle-flan-5ba3b4.netlify.app/

## Key features
- Report issues with title,category,location, description,image,amount and automatic date -- saved to the server with the logged-in user email .

- Authentication using Firebase.

- My Issues -- a table view of issues created by logged-in user.

- My Contribution -- pay or record contributions toward issues and download contribution receipts. 

- Home -- Attractive banner slider , category cards, recent issues etc.

- Responsive design with Tailwind.
